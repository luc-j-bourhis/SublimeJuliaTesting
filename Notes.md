# Implementation notes

* The file .python-version is crucial: by default, SublimeText runs plugins with Python 3.3
  at the time of writing. It can also use Python 3.8, and it is actually the default use for
  the "User" folder. So moving one directory up a Python module in "User" can make it break.
  So my .python-version specifies to use Python 3.8.
